### Range

| 参数     | 说明                                            | 类型    | 默认值 |
| -------- | ----------------------------------------------- | ------- | ------ |
| max      | 最大值                                          | number  | 100    |
| min      | 最小值                                          | number  | 0      |
| step     | 步长，取值必须大于 0，并且可被 (max - min) 整除 | number  | 1      |
| vertical | 是否垂直                                        | boolean | false  |

更多属性请参考 [FormField](/zh/procmp/abstract/field#FormField)。
